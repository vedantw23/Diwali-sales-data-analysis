# Diwali-sales-data-analysis
This project focuses on analyzing Diwali sales data to understand customer purchasing behavior and sales trends during the festive season. The analysis is performed using Python and popular data analysis libraries to extract meaningful insights from the dataset.

The project includes data cleaning, exploratory data analysis (EDA), and visualizations to identify key factors such as top-selling products, high-revenue categories, customer demographics, and sales patterns. These insights can help businesses improve marketing strategies and decision-making during festive periods.

Key Highlights:
Data cleaning and preprocessing
Exploratory Data Analysis (EDA)
Sales and customer behavior insights
Visualizations using graphs and charts

Tools & Technologies Used:
Python
Pandas
NumPy
Matplotlib / Seaborn
Jupyter Notebook
